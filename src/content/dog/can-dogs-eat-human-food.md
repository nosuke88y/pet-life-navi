テーブルで食事中に犬が甘えてくると「少しだけなら…」とつい人間の食べ物を与えてしまいたくなりますよね。しかし、犬と人間では消化能力や代謝が異なるため、人間に無害な食材が犬には毒になることがあります。この記事では、与えてよい食材とダメな食材を一覧で解説します。

## 絶対に与えてはいけない食材（毒性あり）

### 🚫 チョコレート・カカオ製品
テオブロミンという成分が犬には代謝できず、嘔吐・下痢・痙攣・最悪の場合死亡に至ります。ダークチョコレートほど危険性が高い。

### 🚫 タマネギ・ネギ・ニラ・ニンニク
有機チオ硫酸塩が赤血球を壊し溶血性貧血を引き起こします。加熱しても毒性は消えません。少量の継続摂取でも危険です。

### 🚫 ブドウ・レーズン
腎不全を引き起こす成分が含まれています。少量でも危険で、致死量には個体差があるため「少しなら大丈夫」は通用しません。

### 🚫 マカデミアナッツ
後躯麻痺・嘔吐・発熱・震えを引き起こします。致死的ではないことが多いですが、明らかな症状が出ます。

### 🚫 キシリトール（甘味料）
ガム・飴・歯磨き粉などに含まれます。インスリンの過剰分泌による低血糖・肝不全を招きます。

### 🚫 アボカド
ペルシンという成分が心筋障害・呼吸困難を引き起こす可能性があります。

### 🚫 アルコール・カフェイン
微量でも肝臓・腎臓・心臓への毒性があります。コーヒー・お茶・コーラも避けましょう。

## 与えても良い食材（適量で安全）

| 食材 | 注意点 |
|------|--------|
| 鶏肉（茹で・蒸し） | 骨なし・無塩で。生は菌感染リスク |
| 卵（加熱） | 生卵白は避ける（ビオチン欠乏の可能性） |
| 白米・じゃがいも | 無塩でよく加熱する |
| にんじん | 生でも加熱でもOK。低カロリー |
| ブロッコリー | 少量なら可。大量は甲状腺に影響の可能性 |
| りんご（皮なし） | 種・芯は除く。タネにシアン化合物が含まれる |
| バナナ | 糖質が高いので少量に。肥満犬には注意 |
| 茹でかぼちゃ | 消化にもよく、食欲増進にも使える |
| 白身魚（加熱） | 骨なし・無塩で。オメガ3が豊富 |

## 注意が必要な食材（少量・調理次第でOK）

- **牛乳**：乳糖不耐症の犬には下痢の原因に。少量か犬用ミルクを使う
- **豆腐**：大量は消化を乱す可能性。少量なら可
- **きのこ類**：市販のエリンギ・しいたけは加熱すれば可。野生のきのこは禁止

## まとめ

犬に人間の食べ物を与える場合は「チョコレート・タマネギ・ブドウ・キシリトール・アルコール」を絶対に避けることが最優先です。与える前に「犬に安全か？」と一度確認する習慣をつけましょう。誤食した可能性がある場合は、すぐに獣医師に相談してください。
